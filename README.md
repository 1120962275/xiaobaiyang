# 6盘小白羊版
----

### 版权声明：
  
这个程序仅仅是我个人学习而开发，跟6盘官方无任何关联，并不能替代官方PC版功能<br/>
6盘有明确的说明，离线分享链接功能是6pan人工操作，所以离线时间都比较长，并不是故意限制<br/>

### 近期服务不可用说明

因近期6pan调整了分享链接的离线限制，小白羊部分功能暂不可用，我在开发新版，会在3天后更新<br/>

新版功能预告：6盘账户登录，多账户管理，离线到指定账户内，从指定账户下载指定文件等<br/>
